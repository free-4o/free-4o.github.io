2024年最新如何注册ChatGPT教程

准备工作

  1. 邮箱

    邮箱选择海外邮箱，或者自建邮箱，国内的QQ和163..等等基本都被屏蔽

  2. 魔法

    ChatGPT禁止了国内用户的访问，需要魔法上网工具 LeflyCloud、白月光

  3. 国外手机号(可选)

    如果你是开发人员需要用到API那么需要验证手机号，如果非开发人员可以忽略

如果觉得注册麻烦可以直接购买成品号，购买地址：whaleai.cc 成品号都是绑定过海外手机号的。

注册

  1. 打开魔法工具的代理，尽量选择美国节点。（上述推荐的魔法工具，均提供教程）
  2. 打开ChatGPT官网 https://chatgpt.com 点击注册
  3. 输入自己的邮箱地址(请勿使用国内的邮箱,可以使用自建的邮箱)
  4. 然后请根据提示，前往你的邮箱获取验证链接，并点击验证。
  5. 输入你的用户昵称和生日，这里的生日一定要成年！！！！! 获取验证
  6. 然后点击好的，开始吧，继续，在出现的GPT-4o简介，记得一定要点击立即试用（否则此账号以后无法使用免费的4o）
  7. 到此注册就已经结束了，下面的教程针对需要使用API的用户，绑定手机号，获取创建API的权限，不使用API的下面不需要看

绑定手机（可选）

  1. 打开platform,点击右上角的Start verification
  2. 在新的标签页打开 接码平台 点击注册，输入自己的邮箱注册即可(国内邮箱可用)
  3. 注册成功后，点击右上角的 ➕ 充值余额，找到并选择支付宝，最低充值2美金
  4. 充值成功后，返回首页，选择菜单键，输入OpenAI，在下方选择价格升序，选择一个最便宜的号码，点击加购！
  5. 复制号码，打开第一步的标签页，在里面输入手机号，这里需要手动选择号码的国家！！然后把号码粘贴进去，点击Send Code
  6. 如果报错，基本上都是号码问题，返回接码平台进行更换手机号，注意右上角的时间，尽快操作！
  7. 号码可以使用时，会告诉你让你输入验证码，这个时候去接码平台等待验证码即可。
